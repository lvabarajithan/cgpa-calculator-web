Features

CGPA Calculator is an opensource and a cross-platform project initiated to make it available for every students. The reason behind this project can be found on About section. Contribution from various students are welcome, even from you!

If you would like to contribute in some way to the project, see Contribute for more info. This project offers various features...

    Features

    Cross platform User accounts Smart and Accurate Offline functionality Contributions
Cross platform
This project can be experienced in all platforms. Currently the following platforms are supported.
  • Android
  • Web (beta)
  • iOS (currently in development)
  • Windows phone (no support)
User Account

Create a personal account with your unique Anna university number, which enables you to calculate, store and manage your Grade points on the go!

Your updations and changes are synced all across your devices and the desktop, simultaneously with latest cloud backend technology. To know about technology behind, see How it works section.

Smart and Accurate

Calculate your grades quickly and accurately with our efficient algorithm which also enables you to keep track of your arrears (if any) smartly. Your grades can be updated or changed anytime.

This calculator project uses same formula as Anna University uses, so that you get the same and accurate result as in your marksheet.

To know more about updates, see updates section.

Offline functionality

Our cross-platform mobile applications works like charm even when you are offline and all your changes are automatically updated when you get online again. We are planning to add offline functionality even to this website (yes possible!), so that you do not need to have application installed on your mobile everytime to make a new calculation or update existing one.

About

This CGPA Calculator is an Open-source project, meaning anyone (including you!) can edit, modify the entire code & add features to this project and become a contributor. It was just an idea until a student tried to put on his efforts to make it possible. This project was started by him to gain knowledge about how Grade Point Average (GPA) and Cumulative Grade Point Average (CGPA) are calculated and also to learn Android, Web and iOS app development in a single shot!
