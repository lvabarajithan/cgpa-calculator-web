Contribute

Thanks for your interest in contributing to this project. As students, I would like to have you as one of the contributors. If you are willing to contribute and don't know what and how to contribute, then first you must take a look at the below possible contributions..

  • CGPA Calculator for Android
  • CGPA Calculator for iOS
  • CGPA Calculator Website enhancement
  • Spelling mistakes in contents
  • GPA or CGPA Calculation mistakes
  • Bug reporting
  • New feature requesting
  • Features integration

When you are ready to contribute in one of the above mentioned categories, please make sure that you are well-versed in those fields. Example, If you would like to contribute in CGPA Calculator for iOS, then you must have knowledge in Swift programming language. Without the knowledge of the fields for contribution, you cannot contribute.

If you are willing to contribute on non-developmental sections like Bug reporting and GPA or CGPA Calculation miskates, you doesn't require programming knowledge and your contributions will be accepted!

Why Contribute?

This CGPA Calculator is an open-source project meaning anyone, including you, can edit, modify the entire code and try to add features to this project.You can apply your programming skills and knowledge on this project, to make this project far better.

Remember: I'm also one of the student like you. So feel free to contact me about the contribution!

You can quickly send a request for contribution from here.
