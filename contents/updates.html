Updates

This CGPA calculator supports from 2013 regulation, which means that it doesn't cover syllabus of 2008 and previous regulation. This calculator will also be updated once a new regulation has been introduced.

Since this project is open-source, it will recieve continuous updates and improvements from Students community. The current version of the CGPA Calculator is BubbleGum (v2.0.5). Each version will be named by contributors and approved by the project manager. This means that even you can suggest a name for the next version of this project and get your name featured. Note that the name must not contain any harsh contents in indirect means.

To know more about the features of next update release, keep reading.
